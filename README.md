# SpringMvc-CRUD-Application<br>
Developed a full-stack web application using Spring MVC, Spring ORM, Hibernate, and MySQL, implementing complete CRUD (Create, Read, Update, Delete) operations. The application features a responsive and user-friendly interface built with Bootstrap and dynamic content rendering using JSP.

⚙️Technologies Used:<br>
Backend: Spring MVC, Spring ORM, Hibernate<br>
Frontend: JSP, Bootstrap<br>
Database: MySQL<br>
Tools: Eclipse, Tomcat Server<br>
